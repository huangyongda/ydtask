# ydtask
ydtask任务管理 php cli

用来执行php cli 任务
## Installation 安装

```
composer require huangyongda/ydtask
```


###查看正在运行的任务队列命令
```
php demo.php start
```
###查看正在运行的任务队列命令
```
php demo.php status
```
###  use ydtask  用法,请查看demo.php
